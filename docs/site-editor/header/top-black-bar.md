## Onde encontrar

Todas as páginas.

URL de acesso: [lojaarno.myvtex.com/admin/cms/site-editor](https://lojaarno.myvtex.com/admin/cms/site-editor)

## Campos

![image](../../assets/site-editor/top-black-bar-campos.png)

### Link do Blog

Campo para o link de redirecionamento que leva à página inicial do blog Arno.

Exemplo: `/blog`

### Texto do Blog

Campo para o texto que será mostrado no link de redirecionamento do blog.

Exemplo: `Blog Arno.com`

### Link do Bônus

Campo para o link bônus que é renderizado ao lado do link do blog.

Exemplo: `/bonus`

### Texto do Bônus

Campo para o texto do link bônus.

Exemplo: `Bônus`

### Texto da Loja Oficial

Campo para o texto que fica no centro do componente.

Exemplo: `Loja oficial • Tudo em até 6x sem juros • Desconto no PIX`

## Observações

- Os links devem ser relativos à raiz do site. Exemplo: `/bonus`
- Recomenda-se manter os textos curtos para melhor visualização em dispositivos móveis

## Comportamento

- A barra permanece fixa no topo da página
- Os links abrem em uma nova aba
- Cor de fundo: preto (#3b3c4b)
- Cor do texto: branco (#FFFFFF)
